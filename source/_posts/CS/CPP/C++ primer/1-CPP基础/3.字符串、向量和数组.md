## 3.1 命名空间的 using 声明

域操作符

std

```cpp
std::cout;

using namespace std;

cout<<"hello"<<endl;
```

> 头文件不应该包含using声明，因为会被拷贝到引用该头文件的文件中。这可能会造成不必要的麻烦。

## 3.2 标准库类型 string

长度可变的字符序列，需要包含string头文件

### 定义和初始化 string对象

```cpp
#include <iostream>
#include <string>
using namespace std;
int main()
{
    string s1;
    string s2 = s1;
    string s3 = "hello";
    string s4(10, 'c');
    return 0;
}
```



### string 对象上的操作



| 操作      | 描述 |
| --------- | ---- |
| os<<s     |      |
| is>>s     |      |
| getline() |      |
| s.empty() |      |
| s.size()  |      |
| s[n]      |      |
| s1+s2     |      |
| s1=s2     |      |
| s1==s2    |      |
| s1!=s2    |      |
| <,<=,>,>= |      |

> getline得到的string对象中并不包含换行符

* C++语言中，字符串字面值并不是stirng对象，是char[]

```cpp
    string s5=s3+"hello";//正确
    string s6="hello"+"world";//错误error: invalid operands to binary expression ('const char [6]' and 'const char [6]')
```
