# 操作符

在普通模式下



| action |        |                            |
| ------ | ------ | -------------------------- |
| d      | delete | 删除                       |
| c      | change | 修改(删除并进入插入模式)   |
| y      | yank   | 复制                       |
| v      | visual | 选中并进入可视模式(visual) |
| p      | paste  | 复制                       |
| u      | undo   | 撤销                       |
|        |        |                            |



这里记住一个点:

操作符+动作

可以组合出很多的操作,重点在于实践





| action |      |        |
| ------ | ---- | ------ |
| dd     |      | 删除行 |
| yy     |      |        |
|        |      |        |
