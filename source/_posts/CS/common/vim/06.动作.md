# 动作

在普通模式



回顾:

操作符+动作

干什么+怎么做



i:inner

a:around/append



i和a的区别可以简单理解就是包不包含两边的符号

例子:

"test"

i -> test

a -> "test"



| action | 等同 | 解释            |
| ------ | ---- | --------------- |
| iw     |      | word            |
| aw     |      |                 |
|        |      |                 |
| i(     | ib   | 选中            |
| a(     | ab   | (选中)          |
|        |      |                 |
| i"     |      | 选中            |
| a"     |      | "选中"          |
|        |      |                 |
| i'     |      | 选中            |
| a'     |      | '选中'          |
|        |      |                 |
| i<     |      |                 |
| a<     |      | <选中>          |
|        |      |                 |
| i[     |      |                 |
| a[     |      | [选中]          |
|        |      |                 |
| it     |      |                 |
| at     |      | <tag>选中</tag> |
|        |      |                 |
| is     |      |                 |
| as     |      | sentence        |
|        |      |                 |
| ip     |      |                 |
| ap     |      | paragraph       |
|        |      |                 |
| ie     |      | entire          |
