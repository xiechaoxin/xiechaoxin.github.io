# 普通模式的光标移动

hjkl

←↓↑→



| action  |            |                                      |
| ------- | ---------- | ------------------------------------ |
| w       | word       | 跳到下个单词开头                     |
| b       | back/begin | 跳到上个单词开头                     |
| e       | end        | 跳到下个单词结尾                     |
| ge      |            | 跳到上个单词结尾                     |
|         |            |                                      |
| 0       | 零         | 行首                                 |
| ^       |            | 第一个非空字符位置                   |
| $       |            | 跳到行尾                             |
| gg      |            | 跳到第一行                           |
| G       |            | 跳到最后一行                         |
|         |            |                                      |
| f{char} |            | 跳到下个[char]的位置                 |
| F{char} |            | 跳到上个[char]的位置                 |
| t{char} |            | 光标跳到下个{char}的前一个字符的位置 |
| T{char} |            | 光标跳到上个{char}的后一个字符的位置 |
| ;       |            | 重复上次的字符查询命令               |
| ,       |            | 反向查找上次的查询命令               |
