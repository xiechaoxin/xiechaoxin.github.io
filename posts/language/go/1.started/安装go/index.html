Go 环境配置

install go

download go

go version
export GOPATH=$HOME/go

# if you use zsh
source ~/.zshenv

1 添加 Go 工作区文件夹

每个 Go 工作区都包含三个基本文件夹：

bin：包含应用程序中的可执行文件。 src：包括位于工作站中的所有应用程序源代码。 pkg：包含可用库的已编译版本。 编译器可以链接这些库，而无需重新编译它们。

2 运行Go应用

# go run 命令编译并执行应用程序，而不生成二进制文件。 在本地开发和测试应用程序时，将使用此命令
go run main.go
