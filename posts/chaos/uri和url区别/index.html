HTTP 协议中 URI 和 URL 有什么区别？

URI 在于 I(Identifier)是统一资源标示符，可以唯一标识一个资源。URL 在于 Locater，一般来说(URL)统一资源定位符，可以提供找到该资源的路径，比如http://www.zhihu.com/question/21950864，但URL又是URI，因为它可以标识一个资源，所以URL又是URI的子集。举个是个URI但不是URL的例子：urn:isbn:0-486-27557-4，这个是一本书的isbn，可以唯一标识这本书，更确切说这个是URN。总的来说，locators are also identifiers, so every URL is also a URI, but there are URIs which are not URLs.
